# Mobile Preview

A simple, single-file tool to preview any webpage in a mobile device frame.

## Features

- Device presets: Android (360×800), iPhone 14 (390×844), iPhone SE (375×667)
- URL history saved in localStorage (last 10 URLs)
- Persists selected device size across sessions
- Zero dependencies (except optional reboot.css for styling)

## Usage

1. Open `index.html` in a browser
2. Click "→" to enter a URL, or select from history
3. Switch device sizes with the preset buttons

## Installation

Copy `index.html` (and optionally `reboot.css`) anywhere and open in a browser.
